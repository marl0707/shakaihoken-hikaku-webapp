# 社会保険料比較計算ツール

個人事業主向けの社会保険料計算・比較ツールです。

## 機能

- 国民健康保険料の計算
- 国民年金保険料の計算
- 介護保険料の計算（65歳以上）
- 事業主共生連サービスとの料金比較
- 家族構成に応じた計算

## 技術スタック

- React 18
- Vite
- Tailwind CSS
- Framer Motion

## ローカル開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## デプロイ

GitHub Actionsを使用して自動的にGitHub Pagesにデプロイされます。
